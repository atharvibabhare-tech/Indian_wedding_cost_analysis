# Indian_wedding_cost_analysis

## 📌 Overview
This project analyzes wedding data to understand cost patterns based on different factors like wedding type, place, decor, and entertainment.

## 🎯 Objective
- Analyze wedding expenses
- Compare costs across categories
- Visualize data for better understanding

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Analysis Performed
- Data Cleaning
- Groupby Analysis
- Pivot Tables
- Data Visualization (Bar, Pie, Line, Histogram, etc.)

## 🔍 Key Insights
- Wedding cost varies by type and place
- Decor and entertainment increase total cost
- Some categories are more expensive than others

## ✅ Conclusion
This project shows how data analytics helps in understanding patterns and making better decisions.
